Step 2: Create the JS object of the richgrid class

Create our object after the page is ready.
	example = {}
	$(document).ready(function() { 	
		example.grid = new RichHTML.grid({
			el: "example-customer-grid"
		});
		example.grid.render();
	});

Explanation

el:
The RichHTML grid accepts config values when instantiating the object. We will go over all of the available options as we step closer to the final grid. The el config param tells the richgrid which div element to render, place, the grid into.

render()
To display the grid you need to call the object's render() method. This call will load the grid, and performs all it's magic before rendering to the element id you specified with the el property.
