Step 1: Let's load the required JS/CSS files and container element

Richgrid makes use of both jquery and mustache libraries. You will need to include both.

Include the required JS files
			<script type="text/javascript" src="{path}/jquery-1.7.2.min.js"></script>
			<script type="text/javascript" src="{path}/mustache.js"></script>
Include the richgrid JS and CSS files
			<script type="text/javascript" src="richhtml/rich.all-min.js"></script>
			<link rel="stylesheet" type="text/css" href="richhtml/css/rich.grid.css" />

The div should be placed after the <body> and before the </body> tag

Add div container that will hold the richgrid content
	    <div id="tablemetadata"></div>
		<div id="example-customer-grid"></div>

Note: The div tablemetada is optional and only used in the final example to demonstrate showing table paging information.
