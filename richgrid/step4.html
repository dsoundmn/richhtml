Explanation

xtype:
Optional for string variables. The available options are for helper columns.
checkbox: Row select option with dataIndex of the id that identifies your row.
drag: Adds column with draggable icon to allow row reordering. No dataIndex necessary.
expander: Creates a hidden row with additional information (more on this later). dataIndex optional but if you pass it should be the id of the data you want to populate the hidden expander row with.

align: center, left, right - aligns both the column header text and cell values.
width: Column width, when not included 100% is used.
sortable: Adds the column reorder icons. Once the column is clicked the grid is reordered sending to the url action with new fields of sort and dir. Sort will contain the dataIndex name identified in the column description and dir will either be asc or desc.
hidden: defaults to false. When set to true the data will be available for other uses. One example would be to store an id that you want to obtain after sorting columns.
